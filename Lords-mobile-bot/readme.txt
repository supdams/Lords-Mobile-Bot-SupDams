 * Introduction
 This bot aims to :
 - automatize the development of alt account : construction upgrades
 - scan the map for free resource nodes and prepare a sheet that can be found online
 - auto shield on attack. Farm accounts should allow only owner to farm them
 
 * Requirements
 Right now i'm testing it with KoPlayer. In case you want to use some other android emulator, a lot of the search functions might fail.
 Requires the instalation of Autoit if you wish to edit the scripts. Binary format might run without requirements.
 
 * Recommended modules
 Right now autoshield is only working
 
 * Installation
 No special instalation is required for this software. Simply run GeneralBot.exe
 
 * Configuration
 No special configuration is required at the moment
 
 * Troubleshooting
 If pixel search is not working, make sure you run the bot as administrator. Enabling compatibility mode might help also
